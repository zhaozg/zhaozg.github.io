Module callback

callback module for lua-openssl binding

This module provides callback functionality for SSL/TLS operations, including certificate verification callbacks and other SSL event handling.

These callbacks allow customization of SSL/TLS behavior from Lua.

Usage:

    -- Internal module used by SSL module
