Module openssl

lua-openssl binding, provide openssl base function in lua.

Usage:

    openssl = require('openssl')

error ([error[, clear=false]])
get last or given error infomation Most lua-openssl function or methods return nil or false when error or failed, followed by string type error reason and number type error code, code can pass to openssl.error() to get more error information.

Parameters:

  • error number , default use ERR_get_error() return value (optional)
  • clear boolean the current thread’s error queue. (default false)

Returns:

  1. number errcode
  2. string reason
  3. string library name
  4. string function name
  5. boolean is this is fatal error
